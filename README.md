# Emotion Based Music Recommendation System

This is used to generate music recommendation as per human mood or emotions, It is a web app whose backend host a trained model and runs in flask. You can run this project on your local machine as well as on cloud very easily.


# Environment Setup

Make sure Anaconda is installed and launch anaconda prompt and navigate to root directory in the anaconda prompt

create venv

```shell
conda create -n "emotionbasedmusicrecommendation" python=3.10
```

Activate

```shell
conda activate emotionbasedmusicrecommendation
```

In order to set your environment up to run the code here, first install all requirements:

```shell
pip install -r requirements.txt
```

run the app.py file 

```shell
python app.py
```

Once you see this url - http://127.0.0.1:8080/ in logs, open it in browser.
